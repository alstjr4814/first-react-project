🅐 A. VSCode & 개발환경 정리
1. VSCode 주요 단축키

Ctrl + Shift + P : VSCode 검색창(명령 팔레트)

Ctrl + Space : 자동 import / 자동완성

Alt + Shift + F : 코드 자동 정렬(자동 들여쓰기)

2. React 개발 실행 관련

npm run dev : React(Vite) 개발 서버 실행

clone 받은 프로젝트는 node_modules 없음 → 반드시

npm install
npm run dev

3. 파일/폴더 운영 팁

React 프로젝트는 프로젝트 루트(folder) 를 열어야 함
(예: first-react-project 전체를 열기)

input이 안되거나 오류가 반복되면:

폴더 copy

내용 붙여넣기

기존 폴더 삭제

copy 이름 원래대로 변경

VSCode 재시작

4. VSCode 설정 팁

JSX snippet 설정:
File → Preferences → Configure → JSX에서 rsf 자동 생성 가능
→ 컴포넌트 기본 구조 자동 생성
